13//! A library that provides ASCII-only string and character types, equivalent to the `char`, `str`
14//! and `String` types in the standard library.
15//!
16//! Please refer to the readme file to learn about the different feature modes of this crate.
17//!
18//! # Requirements
19//!
20//! - The minimum supported Rust version is 1.9.0
21//! - Enabling the quickcheck feature requires Rust 1.12.0
22//! - Enabling the serde feature requires Rust 1.13.0
23//!
24//! # History
25//!
26//! This package included the Ascii types that were removed from the Rust standard library by the
27//! 2014-12 [reform of the `std::ascii` module](https://github.com/rust-lang/rfcs/pull/486). The
28//! API changed significantly since then.
29
