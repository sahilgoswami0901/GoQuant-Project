Get vault balance.

Check your vault balance (total, locked, and available).

§Endpoint

GET /vault/balance/{user}

§Example

curl http://127.0.0.1:8080/vault/balance/YOUR_WALLET_ADDRESS

§Path Parameters

  • user - User’s wallet public key (base58)
